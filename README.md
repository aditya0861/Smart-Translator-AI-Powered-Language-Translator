# Smart Translator

A minimal and elegant language translator web application built using modern frontend technologies.

## Features

- Real-time language translation  
- Premium warm-beige UI theme  
- Smooth typing animation  
- Micro-animations with Framer Motion  
- Responsive design  
- Clean and minimal interface  

## Tech Stack

- React  
- Tailwind CSS  
- Framer Motion  
- Axios  
- Google Translator API  

## Setup

1. Clone the repository  
2. Install dependencies  
   npm install  
3. Add your RapidAPI key in .env  
4. Run the project  
   npm run dev
